# 📌 Validador de CPF  

Este projeto em **Python** verifica se um CPF é válido ou inválido, seguindo as regras do **dígito verificador**.

## 🛠️ Tecnologias Utilizadas  
✅ **Python** 🐍  
✅ **Unittest**
